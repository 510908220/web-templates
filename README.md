## 注意

- 安装`nodejs`: https://nodejs.org/en/download/package-manager/
- 安装cnpm: 国内的话可以使用淘宝定制的cnpm (gzip 压缩支持) 命令行工具代替默认的 npm,这样不仅速度快避免由于下载一些依赖导致失败.
